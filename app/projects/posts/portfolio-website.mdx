# Personal Portfolio Website

A comprehensive portfolio website showcasing my work, research, and technical interests. Built with modern web technologies and optimized for performance and accessibility.

## Design Philosophy

The website follows a minimalist design approach with:

- **Dark Theme**: Professional appearance suitable for technical content
- **Typography**: Space Mono font for a clean, monospace aesthetic
- **Responsive Design**: Seamless experience across all device sizes
- **Performance First**: Optimized loading times and smooth interactions

## Technical Implementation

### Framework Choice

Built using Next.js 15 with the App Router for:
- Server-side rendering for better SEO
- Static site generation for optimal performance
- Built-in optimization for images and fonts
- Excellent developer experience

### Content Management

- **MDX Integration**: Rich content with embedded React components
- **Dynamic Routing**: Automatic page generation for blog posts and projects
- **Metadata Handling**: Comprehensive SEO optimization
- **Syntax Highlighting**: Code blocks with syntax highlighting support

### Styling Approach

Implemented with Tailwind CSS for:
- Utility-first CSS approach
- Responsive design patterns
- Dark mode support
- Consistent spacing and typography

## Features

### Blog System

- MDX-powered blog with frontmatter metadata
- Syntax highlighting for code blocks
- Math equation support with KaTeX
- Responsive typography with prose styles

### Project Gallery

- Grid-based layout showcasing projects
- Technology tags and status indicators
- Featured project highlighting
- Detailed project pages with rich content

### Performance Optimizations

- **Image Optimization**: Next.js Image component with lazy loading
- **Font Optimization**: Google Fonts with display swap
- **Bundle Analysis**: Optimized JavaScript bundles
- **Lighthouse Score**: 100/100 performance score

## Analytics & Monitoring

Integrated analytics and monitoring:

- **Vercel Analytics**: Real-time performance insights
- **Speed Insights**: Core Web Vitals monitoring
- **SEO Optimization**: Structured data and meta tags

## Deployment

Deployed on Vercel with:

- **Automatic Deployments**: GitHub integration for CI/CD
- **Edge Functions**: Global content delivery
- **Custom Domain**: Professional domain configuration
- **SSL Certificates**: Automatic HTTPS encryption

## Future Enhancements

Planned improvements include:

- Search functionality for blog posts and projects
- Contact form with email integration
- Newsletter subscription system
- Interactive resume/CV section
