# rqho.github.io

My personal website built with Next.js, React, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when pushing to the `main` branch.

Live at: [https://rqho.github.io](https://rqho.github.io)
